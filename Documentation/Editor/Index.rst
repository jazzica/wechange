===========
For Editors
===========

Target group: **Editors**

Frontend Plugins
========

The extension offers two frontend plugins.

.. tip::

   If you are unsure how to add a frontend plugin to your TYPO3 site please refer to the official TYPO3 documentation on
   :ref:`t3editors:content-creating`.

Project List
------------
The Project List frontend plugin displays a list of projects. The list can be reduced or extended by

* specifying a group id
* specifying a tag
* setting the project result limit (default: 10)

Map
---
The Map frontend plugin embeds the WECHANGE map in an iFrame. The map can be configured as follows

* show or hide people
* show or hide events
* show or hide projects
* show or hide groups
* show or hide ideas
* filter by group id
* specifying a result limit (default: all entries matching the given criteria)
