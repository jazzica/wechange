===========
For Editors
===========

Target group: **Editors**

Frontend Plugins
================

The extension offers the following frontend plugins:

*  Project List
*  Event List
*  Conference List
*  Notes List
*  Statistics
*  Map

.. tip::

   If you are unsure how to add a frontend plugin to your TYPO3 site please refer to the official TYPO3 documentation on
   :ref:`t3editors:content-creating`.
