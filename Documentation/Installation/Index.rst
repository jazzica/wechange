============
Installation
============

Target group: **Administrators**

The extension is installed like any other standard TYPO3 extension. There are no dependency to other extensions or services.

.. tip::

   Please make sure to include the static TypoScript template. Don't know how? Please see :ref:`t3sitepackage:extension-installation`

If you need any help installing a TYPO3 extension please see the official TYPO3 documentation for support:
:ref:`t3install:extension-installation`.
