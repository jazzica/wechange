============
Installation
============

Target group: **Administrators**

The extension is installed like any other standard TYPO3 extension.

.. code-block:: bash

   composer req jazzica/wechange

Dependencies
================

*  PHP >= 7.4.0 <= 8.1.99
*  ext-json and ext-curl
*  There are no dependency to other TYPO3 extensions

.. tip::

   Please make sure to include the static TypoScript template. Don't know how? Please see :ref:`t3sitepackage:extension-installation`
